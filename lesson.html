Module 1: Introduction

Understanding Variables in Python

Variables are fundamental to programming. Think of them as containers for storing data values.

In Python, you don't need to explicitly declare the type of a variable. The declaration happens automatically when you assign a value to it. The equals sign (=) is used to assign values to variables.

Example:

Here, we create a variable named `course_name` and assign it the string value "BitMentor Python Course". We also create a variable `students` and assign it the number 50.

course_name = "BitMentor Python Course"
students = 50

print(course_name)
print(students)

Naming Rules

A variable name must start with a letter or the underscore character. It cannot start with a number and can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ ). Variable names are case-sensitive (`age`, `Age` and `AGE` are three different variables).
